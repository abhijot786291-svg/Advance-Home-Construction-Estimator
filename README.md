# Advance Home Construction Estimator

A Python-based civil engineering cost estimation project.
Author : Abhijot 
## Features

- Plot Area Calculation
- Built-up Area Calculation
- Brick Estimation
- Cement Estimation
- Sand Estimation
- Steel Estimation
- Concrete Estimation
- Doors, Windows & Ventilators
- Woodwork Estimation
- Electrical Estimation
- Plumbing Estimation
- Boundary Wall & Gate Estimation
- Labour Cost Estimation
- Material Summary
- Cost Breakdown
- Cost Per Square Foot

## Technology Used

- Python 3

## How to Run

```bash
py Advance_Home_Construction_Estimator.py
